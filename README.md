# core

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

This project was created using `bun create tui`. [create-tui](https://git.new/create-tui) is the easiest way to get
started with OpenTUI.

## Build

### MacOS

```
bun build ./src/index.ts --compile --outfile my-cli-macos
```

### Linux

```
bun build ./src/index.ts --compile --outfile my-cli-linux
```

### Windows

```
bun build ./src/index.ts --compile --outfile my-cli-windows.exe
```
